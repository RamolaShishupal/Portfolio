First Django App

  

Hi friends, Today I will tell u guys how you can make an web app using Django in minimal steps so Let’s get started…

Step No 1: Install Django on your system its very easy for linux users just put following cammands in your terminal

sudo apt-get install pip
# this will install pip in your system

python install Django
# this will install django in your system

Step No 2: Now make a virtual environment for your project directory so you could put all your code in it

Step No 3: In your Project Directory You should make a Template Folder so Yould put Your Front end code in it like your HTML AND CSS code that is your user interface

Step No 4: Add your web app name in Settings.py of main project

Step No 5: Connect Your Front end with Views :

# Create your views here.
def MyfirstApp(request):
    return render(request, "homepage.html")

Step No 6: Now You have done with youe front ens part so to call your webpage u need to add followith path in urls.py of your main project folder as well as your app Folder:

from django.urls import path

from . import views

urlpatterns = [
    path("homepage.html", views.MyfirstApp, name="homepage"),
]

Step No 7: Put your logic part in Models.py of your app Folder

from django.db import models
from django.contrib.gis.db import models


# Create your models here.
class Multiplication(models.Model):
    FirstNumber = models.FloatField()
    SecoundNumber = models.FloatField()

    def __str__(self):
        return self.location

Step No 8: Migrate your app using following cammand:

python manage.py make migrations
# this cammand will Migrate your code on localhost

python manage.py migrate
# this cammand will run your script that is your app so you can see your output

Thats it this are some essential steps for any basic web app in django so now you people have to explore more in it and emjoy making wen applications in django becouse its super facinating and fast so Let’s code..
